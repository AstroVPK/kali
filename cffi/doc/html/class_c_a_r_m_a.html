void CARMA::operator() ( const vector< double > &  x,
vector< double > &  dxdt,
const double  t 
)

Compute and return the first column of expm(A*dt)*B*trans(B)*expm(trans(A)*dt)

At every step, it is necessary to compute the conditional covariance matrix of the state given by \(\textbf{\textsf{Q}} = \int_{t_{0}}^{t} \mathrm{e}^{\textbf{\textsf{A}}\chi} \mathbfit{B} \mathbfit{B}^{\top} \mathrm{e}^{\\textbf{\textsf{A}}^{\top}\chi} \mathrm{d}\chi\). Notice that the matrix \(\textbf{\textsf{Q}}\) is symmetric positive definate and only the first column needfs to be computed.
